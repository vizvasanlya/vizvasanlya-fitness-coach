Plan workouts, track training consistency, monitor recovery, and manage fitness goals.

## Features

- **Workout Planning** — Create and schedule training sessions
- **Consistency Tracking** — Monitor workout frequency and streaks
- **Recovery Monitoring** — Track rest days and recovery status
- **Goal Management** — Set fitness targets and track progress
- **Search & Filter** — Find workouts by type, status, or date
- **Analytics View** — Training distribution and consistency insights
- **Import / Export** — JSON-based fitness data portability

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES2022) |
| Storage | LocalStorage |
| Server | Python HTTP server (dev) |

## Getting Started

### Prerequisites

- Python 3.x (for dev server)
- Modern web browser

### Installation

```bash
git clone https://github.com/vizvasanlya/First.git
cd First/vizvasanlya-fitness-coach
```

### Run

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173)

### Docker

```bash
docker compose up --build
```
